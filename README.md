# Personal Portfolio CLI App

## How to Run The App
1. Ensure Python is installed on your system
2. Download the `portfolio-cli-app.py` file
3. Open terminal/command prompt
4. Navigate to the file location
5. Run: `python portfolio-cli-app.py`

## What I Learned
- How to structure a CLI application
- Input validation and error handling
- Using functions to organize code
- Git workflow and documentation

## Challenges I Faced
- Handling user input errors
- Organizing code into logical functions
- Learning how to merge unrelated histories
